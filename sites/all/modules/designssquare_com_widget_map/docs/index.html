Installation

Here are steps to install Maps widget:

  1. Enable "DesignsSquare Maps Widget" for any Maps to work

      For Elastic Maps

    1. Enable "DesignsSquare Elastic Maps Slide" which also will enable "DesignsSquare.com: Maps: Elastic: Structure"
    2. For importing Sample Data, enable "DesignsSquare.com: Maps: Elastic: Sample Data"
    3. Assign Elastic Maps block to a region you like the Maps to be displayed. If sample data was imported, then there is already block "DesignsSquare.com: Maps: Elastic" to be assigned. Otherwise, create a view with tag - designssquare_Maps_elastic as described in the "How To" section below for the Elastic Maps

      For Revolution Maps

    1. Enable "DesignsSquare Rev Maps" which also will enable "DesignsSquare.com: Maps: Revolution: Structure"
    2. For importing Sample Data, enable "DesignsSquare.com: Maps: Revolution: Sample Data"
    3. Assign Rev Maps block to a region you like the Maps to be displayed. If sample data was imported, then there is already block "DesignsSquare.com: Maps: Rev Maps" to be assigned. Otherwise, create a view with a tag - designssquare_Maps_rev_Maps as described in "How To" for Revolution Maps

Troubleshooting

1. The file used in the Image field may not be referenced

This error came up when the sample data import didn't added entry in table 'file_usage' when transfering artifacts. Make sure you have the newest version of the DesignsSquare.com Maps Widget
